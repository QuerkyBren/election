FAQ

How do you know what nation I am?

NationStates allows for third-party websites to verify whether or not you are logged in as a certain nation, using a special access code. We do not get any access or information of your nation besides your login status.

How are the votes accountable?

All vote records are logged using Firebase, a data storage service for web apps by Google. Firebase frequently does backups on vote data, as well as provides us with an unmodifiable log of vote changes (through citizens voting, decisions made by the Minister of Justice or new elections created by government officials). This log records which user did what at a certain time and from what IP address.

Additionally, all of the code of the website that facilitates this database and website is open source and can be viewed on GitHub. Changes to the code by any developer are recorded there using git, and the history of changes can not be modified.

Why is a new site necessary?

A specialized website designed for Versutian elections ensures that the voting process is accountable, accessible and easily managed. Although forums do provide for polling functionality, more features to the voting process can be added through a special site.
